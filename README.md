# BrickBreaker (Unity)

A simple **Brick Breaker / Breakout-style** game built in **Unity**. Control the paddle to keep the ball in play and clear all bricks to win.

## Gameplay
- Move the paddle left/right to bounce the ball
- Break bricks to score points
- Lose a life if the ball falls below the paddle
- Clear all bricks to complete the level

## Controls
- **A / D** or **Left / Right Arrow**: Move paddle  
- *(Optional)* **Mouse**: Move paddle (if enabled in your build)

## Built With
- **Unity** (project includes standard Unity folders: `Assets/`, `Packages/`, `ProjectSettings/`)
- **C#** scripts inside `Assets/`

## How to Run
### Option 1: Open in Unity (recommended)
1. Install a compatible Unity version using **Unity Hub**
2. In Unity Hub, click **Add** and select this repository folder
3. Open the project
4. Press **Play** in the Unity editor
